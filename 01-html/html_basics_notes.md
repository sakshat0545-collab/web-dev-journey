# HTML Basics as a Beginners 🚀

HTML (HyperText Markup Language) is the standard markup language used to create webpages and web applications.

It provides the basic structure of a webpage using different elements and tags.

---

# Why Learn HTML?

HTML is the foundation of web development.

Every website on the internet uses HTML in some form.

Learning HTML helps in:
- Creating webpage structure
- Building forms and layouts
- Understanding frontend development
- Working with CSS and JavaScript

---

# Basic Structure of an HTML Document

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>

<body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>
```

---

# Important HTML Tags

## Heading Tags

Used to define headings on a webpage.

```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Smaller Heading</h3>
```

---

## Paragraph Tag

Used to write paragraphs.

```html
<p>This is a paragraph.</p>
```

---

## Anchor Tag

Used to create links.

```html
<a href="https://google.com">Visit Google</a>
```

---

## Image Tag

Used to display images.

```html
<img src="image.jpg" alt="Sample Image">
```

---

## List Tags

### Ordered List

```html
<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ol>
```

### Unordered List

```html
<ul>
    <li>Python</li>
    <li>Django</li>
</ul>
```

---

# HTML Attributes

Attributes provide additional information about HTML elements.

Example:

```html
<a href="https://github.com">GitHub</a>
```

Here:
- `href` is an attribute
- it specifies the link destination

---

# Semantic HTML

Semantic tags improve readability and SEO.

Examples:
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<footer>`

Example:

```html
<header>
    <h1>My Website</h1>
</header>
```

---

# Forms in HTML

Forms are used to collect user input.

Example:

```html
<form>
    <input type="text" placeholder="Enter your name">
    <button>Submit</button>
</form>
```

---

# HTML Best Practices

- Use proper indentation
- Write semantic HTML
- Keep code clean and readable
- Use meaningful tag names
- Always add `alt` text for images

---

# Beginner Mistakes to Avoid

- Forgetting closing tags
- Improper nesting of elements
- Using too many `<br>` tags
- Ignoring semantic tags
- Writing unorganized code

---

# Next Step After HTML

After learning HTML:
1. Learn CSS for styling
2. Learn JavaScript for interactivity
3. Build mini projects
4. Start React development
